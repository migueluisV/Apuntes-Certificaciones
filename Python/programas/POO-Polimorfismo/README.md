Esta carpeta contiene dos subcarpetas y un archivo, los cuales son:

- `main.py`: es el código fuente del programa.

## Descripción del programa

El objetivo de este programa es repasar sobre la **programación orientada a objetos**, en específico, sobre el 
**polimorfismo**. No es más que un método, de una clase base a sus clases derivadas, tenga un comportamiento distinto 
en cada una de ellas, este concepto involucra también la **herencia**, podemos apreciar que la forma de declarar los 
métodos donde se aplicará el polimorfismo es la misma que si utilizaramos la abstracción, pero no involucramos el uso 
de la clase _ABC_ ni el decorador _abstractmethod_.

El programa solicita al usuario cuatro nombres para cuatro tipos de animales: aves, peces, mamíferos y reptiles, estos 
nombres son mandados a respectivos objetos de clase de cada tipo de animal, y se llama al método de desplazarse, el 
cual despliega el nombre del animal, y su forma de desplazarse.
