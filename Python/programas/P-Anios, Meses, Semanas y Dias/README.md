## Bienvenidos

Este programa fue realizado en PyCharm Community Edition 2022.1.

## Estructura de la carpeta

Esta carpeta contiene dos subfolders y un archivo, los cuales son:
- `.idea`: aquí se almacenan todas las configuración del proyecto en archivos *.xml*.
- `main.py`: es el código fuente del programa.

## Descripción del programa

El objetivo de este programa es repasar sobre las clases brevemente, sobre sus métodos y atributos, a la par que 
utilizamos la simulación de un ciclo _do-while_ con la instrucción **break** para seguir ejecutando el programa hasta que el usuario dicte que no.

El programa recibe un número entero, el cual es tratado como una cantidad de días transcurridos, de este número se 
busca saber cuantos años, meses, semanas y días restantes hay en él. Se utiliza una clase que recibe en su constructor 
el número entero de días transcurridos, se calculan los datos de este número utilizando el operador **modulo** (**%**) 
y finalmente se despliegan estos valores.
