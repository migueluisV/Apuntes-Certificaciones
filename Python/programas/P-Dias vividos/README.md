## Bienvenidos

Este programa fue realizado en PyCharm Community Edition 2022.1.

## Estructura de la carpeta

Esta carpeta contiene dos subfolders y un archivo, los cuales son:
- `.idea`: aquí se almacenan todas las configuración del proyecto en archivos *.xml*.
- `main.py`: es el código fuente del programa.

## Descripción del programa

El objetivo de este programa es repasar sobre la sintáxis básica de Python, como lo es 
la declaración de funciones, tanto la que regresa un valor, como la que no, un poco sobre 
bloques try:-except: y el uso de la función *format()* para cadenas.

El programa simplemente recibe un número entero, que es utilizado como la edad de una 
persona, y se regresa la cantidad de días vividos de dicha persona.
