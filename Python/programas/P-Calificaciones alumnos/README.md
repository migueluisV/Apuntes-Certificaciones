## Bienvenidos

Este programa fue realizado en PyCharm Community Edition 2022.1.

## Estructura de la carpeta

Esta carpeta contiene dos subcarpetas y un archivo, los cuales son:
- `.idea`: aquí se almacenan todas las configuraciones del proyecto en archivos *.xml*.
- `main.py`: es el código fuente del programa.

## Descripción del programa

El objetivo de este programa es repasar sobre la sintaxis básica de Python, resaltando la creación de clases con su 
constructor donde se **inicializan los distintos atributos** que esta clase posee, por medio de parámetros y asignación 
de parámetros a atributos, cabe recalcar que aparece la **combinación de condiciones en una sola**, **asignación de un mismo valor a múltiples variables**, **agregado de 
elementos a listas por entrada del usuario**, **simulación de ciclo _do-while_**, **redondeo de decimales** y 
**despliegue de múltiples líneas en una sola instrucción _print_**.

El programa trabaja con dos listas, en una se agregan las calificaciones de alumnos y en la otra los nombres de los 
mismos, y se obtienen los siguientes datos: calificación mayor, menor, aprobatorias, reprobatorias, válidas (entre 0 y 
10), reprobatorias (menor a 0 y mayor a 10) y promedio, para finalmente desplegar esta información.
