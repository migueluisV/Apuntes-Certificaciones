## Descripción del programa

El objetivo de este programa es repasar sobre la **programación orientada a objetos**, en específico, sobre los 
**constructores** y **destructores**. Los constructores son llamados cuando un objeto de clase es creado, y los 
destructores son llamados cuando este objeto es eliminado, termina el programa o no hay más referencias del objeto, 
estos términos son utilizados creando los métodos _init_ y _del_, en este código, quisimos utilizar clases abstractas 
solamente para ejemplificar el uso de constructores y destructores, sin embargo, no es necesario el uso de la 
abstracción.

El programa busca calcular el perímetro y área de un triángulo y rectángulo, se reciben valores para las bases y 
alturas de estas figuras y se calculan los resultados.
