## Bienvenidos

Este programa fue realizado en PyCharm Community Edition 2022.1.

## Estructura de la carpeta

Esta carpeta contiene dos subfolders y un archivo, los cuales son:
- `.idea`: aquí se almacenan todas las configuración del proyecto en archivos *.xml*.
- `main.py`: es el código fuente del programa.

## Descripción del programa

El objetivo de este programa es repasar sobre la sintáxis básica de Python, como lo es la declaración de funciones, 
en especial las que no regresan un valor, un poco sobre bloques try:-except: y el uso de la función format() para 
cadenas.

El programa simplemente recibe un número, que representa una edad, y despliega la cantidad de meses transcurridos en 
dicha edad, o la cantidad de días trancurridos en esa edad.
