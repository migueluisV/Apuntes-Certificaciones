## Descripción del programa

El objetivo de este programa es repasar sobre los tipos de sorteos más populares, en este caso: **el sorteo burbuja**. 
Los ordenamientos de colecciones de datos se basan en comparar los elementos entre sí, sin embargo, existen distintas 
formas de compararlos, el método burbuja compara el elemento en el índice _i_ con el que tiene el índice _i + 1_; este 
método tiene tres variantes que vuelven la comparación más rápida, en este programa, vemos los tres tipos y podemos 
escoger si deseamos ordenarlos ascendente o descendentemente.

El programa solicita al usuario la cantidad de elementos de la lista, posterior a ello, ingresamos dicha cantidad de 
elementos en la lista, los métodos de ordenamiento regresan la lista ordenada, por lo que a una lista nueva le asignamos 
el la lista ordenada que regresa el método seleccionado, finalmente, despliegan la lista ordenada.
