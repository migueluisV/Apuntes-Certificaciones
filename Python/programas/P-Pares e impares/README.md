## Estructura de la carpeta

Esta carpeta contiene dos subcarpetas y un archivo, los cuales son:
- `.idea`: aquí se almacenan todas las configuraciones del proyecto en archivos *.xml*.
- `main.py`: es el código fuente del programa.

## Descripción del programa

El objetivo de este programa es repasar sobre las sintaxis básica de Python, contempla uno de los ejercicios de 
programación más simples: determinar si un número es par o impar con el uso del operador módulo (**%**), seguimos 
aplicando el uso de _simulación de ciclo do-while_, y el uso de _bloques try-catch-finally_.

El programa simplemente recibe un número entero, utilizando una función que determina si es par o impar, completa 
la cadena resultado que vemos en pantalla, una vez tenemos nuestro resultado, se nos pregunta si queremos ver si otro 
número es par o impar.
